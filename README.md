
## Description

This Prject Assignment is a Crud operation on a user record which will perform Create,Edit,Delete,Find Operation on a user Record.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Support
This Poject assignment is created on Nest.js, Please ensure system having installed Node.js
